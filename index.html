📤 Add a New Video

Two ways to add a video. Paste a video URL (from Microsoft Stream, YouTube, Vimeo, or any embeddable service) — this is the recommended approach. Or upload a video file directly for quick local testing — note that direct uploads only work in this browser session and aren't saved long-term.

Paste the share link from YouTube, Microsoft Stream, Vimeo, or any video host. For YouTube, use the standard watch link.

Tags help Opti and search find this video. Be generous — more tags = easier to discover.

📖 Quick Tips

Where should I host my video files?

The simplest option: upload to Microsoft Stream (part of your Office 365) and paste the share URL here. YouTube Unlisted works too — the video won't appear in public search but anyone with the link can watch. For quick demos you can drop a file directly, but those won't persist when you close the browser.

How do categories work?

Tech Skills = tools and platforms (Engage, Teams, Blender). Pedagogy Skills = teaching frameworks (UDL, PICRAT). Example Lessons = teacher showcases of actual VR and AI lessons — filtered by subject and grade.

One-offs vs. Courses

A standalone video ("How to invite subs to a Teams meeting") is a one-off. If you're building a multi-video deep dive on one topic, create a course first in the Manage Courses tab, then assign videos to it. Course videos still appear in search and category rows.

Testing with your 3 sample videos

Try adding:
• "How To Add OAO Decals to VR Avatar" → Tech Skills / Engage / one-off
• "How to Invite & Add Subs as Co-Organizers" → Tech Skills / Microsoft Teams / one-off
• "How To Keyframes" → Tech Skills / Experience Editor / assign to an Experience Editor course

📚 How OptimaPD Works

How data is saved

In local testing mode, videos and courses are saved in this browser's localStorage — a small storage area every web browser has built in. Data persists across reloads but doesn't sync across devices. In cloud mode (Firebase configured), data lives in a Firestore database that syncs live across every device, browser, and user.

Migrate local data to Firebase

Already added videos in local mode and now have Firebase configured? Push them all to the cloud database in one click.

Resetting / clearing

Want to start fresh? Click the button below. This will clear all videos and courses you've added.

Export / Backup

Download your entire library as a JSON file for safekeeping or to move to another browser.

Video URL tips

YouTube: paste the standard watch URL (like https://youtube.com/watch?v=abc123). It'll automatically embed.
Microsoft Stream: use the "share" link from Stream.
Direct MP4 files (hosted anywhere): paste the direct URL ending in .mp4 or .webm.
